# kufotoin
Website portofolio (Premium) for Kufotoin_
Contents:
- index.html
- style.css
- script.js
- assets/ (place your photos here: photo1.jpg ... hero.jpg ... favicon.png ... og-image.jpg)

## Deploy to GitHub Pages
1. Create a repository named `kufotoin` on your GitHub account.
2. Upload all files (do not upload the parent folder; upload the files inside).
3. In repo Settings -> Pages -> Source: Branch `main` (or `master`) -> / (root) -> Save.
4. Wait a few minutes, site will be available at: https://<your-username>.github.io/kufotoin

## Customize
- Replace assets/* with your photos.
- Edit contact email in index.html.
